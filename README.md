# Gacha Probability Analyzer

📱 輕鬆管理你的抽卡清單！超直覺的機率分析工具，專為遊戲玩家打造。

無論是要販售道具、贈與朋友，還是要整理自己的清單，都能輕鬆搞定！不用再手忙腳亂地一個個打字了～

https://hinake27.github.io/Cope_mabinogi_text/

## 功能特點

- 📊 機率表解析
  - 支援 SABC 等級分類
  - 自動計算總機率
  - 即時機率驗證

- 🔍 道具搜尋
  - 即時搜尋過濾
  - 顯示詳細機率資訊
  - 快速添加到記事本

- 📝 記事本功能
  - 自動按等級排序
  - 可獨立修改道具數量
  - 一鍵複製清單
  - 支援批量清除

## 使用方法

1. 複製遊戲中的機率表，貼到左側輸入框
2. 系統會自動解析並顯示道具列表
3. 使用搜尋功能找到需要的道具
4. 點擊 `+` 添加到記事本
5. 在記事本中可以：
   - 修改每個道具的數量
   - 移除不需要的道具
   - 複製整個清單
   - 一鍵清除所有記錄

## 支援的機率表格式

```
S等級道具名稱 1.000%
A等級道具名稱 2.000%
B等級道具名稱 3.000%
C等級道具名稱 4.000%
一般道具名稱 5.000%
```

也支援不帶等級的一般格式：
```
道具名稱 1.000%
```
